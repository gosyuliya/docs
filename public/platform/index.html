    Платформа

    Платформа «Цифриум» — это онлайн-сервис, объединяющий различные веб-ресурсы в единое пространство для доступа к образовательному и просветительскому контенту.

    Платформа состоит из 11-ти продуктовых и 2-х тестовых инстансов.

    Основные сведения о Платформе#

    Перечень инстансов#

    staging — цифровая платформа для тестирования нового функционала.

    rc — тестовая предпродуктовая цифровая платформа.

    Образовательные проекты Цифриум (edu) — это образовательные проекты, которые относятся к b2b сектору. На инстансе проходят школьные и муниципальные этапы Всероссийской олимпиады школьников для регионов России.

    Онлайн-олимпиады в Московской области (mo) — это образовательные проекты для Московской области (Взлет.Онлайн, школьные и муниципальные этапы Всероссийской олимпиады школьников). Особенность платформы заключается в том, что дети авторизуются на инстансе через сервис «Моя школа».

    ПРО ЕГЭ (ege) — цифровая платформа для подготовки к государственным экзаменам.

    БАС УНТИ (unti) — цифровая платформа, созданная в рамках федерального проекта «Кадры для БАС» национального проекта «Беспилотные авиационные системы».

    Код будущего (kb) — цифровая платформа, созданная в рамках федерального проекта «Код будущего». Это курсы, на которых школьники 8—11 классов и студенты техникумов и колледжей изучают программирование.

    Дополнительное образование (do) — современная цифровая среда для организации развивающих кружков (дополнительных занятий) в школе.

    Олимпиада Иннагрика (inn) — Всероссийская олимпиада по агрогенетике для школьников 9-11 классов «Иннагрика».

    Инженеры будущего (kpt) — цифровая платформа для курсов Технопарка Физтехлицея им. Капицы (МФТИ).

    Оператор БВС (dksta) — цифровая платформа unti для провайдера из Владимира.

    Технолицей (itech) — IT-курсы Технолицея им. В. И. Долгих.

    Домашняя школа «Цифриум» (homeschool) — цифровая платформа для домашнего обучения школьников.

    Компоненты и инструменты Платформы#

    Для работы Платформы интегрированы сторонние виджеты:

    • ULMS
    • Кинескоп
    • Mind Box
    • Рассыльщики
    • Яндекс Метрика
    • Meta Base

    Микросервисы для бизнес-процессов#

    Для автоматизации и поддержки бизнес-процессов используются специализированные микросервисы:

    • CVS
    • Тест Про
    • МКБ
    • МБАС
